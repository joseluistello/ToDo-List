## 3. How to run

### 3.1 Installation

1. Clone the repo
   ```py
   git clone https://github.com/joseluistello/ToDo-List.git
   ```

2. Create Virtual Enviroment
   ```py
   cd ToDoList
   ```
   ```py
   virtualenv venv
   ```
   
3. Run Virtual Enviroment
   ```py
   venv\Scripts\activate
   ```
   
4. Install Django
   ```py
   pip install Django
   ```

5. Run Django 
   ```py
   python manage.py runserver
   ```
